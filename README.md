## Getting Started

```bash
# Clone this repo
git clone https://github.com/singaravelan/adk-linkedin-series.git
cd adk-linkedin-series

# Install ADK
pip install google-adk

# Jump into any week's folder and follow the instruction.md
cd week1_intro_to_adk
cat instruction.md
```
